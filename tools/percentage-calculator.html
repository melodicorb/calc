Understanding Percentages

Basic Percentage

A percentage is a number expressed as a fraction of 100.

X% = X/100

Example: 25% = 25/100 = 0.25

Percentage of a Number

To find X% of Y:

Result = (X/100) × Y

Example: 20% of 150 = (20/100) × 150 = 30

Percentage Change

The percentage increase or decrease from one value to another.

% Change = ((New - Original) / |Original|) × 100

Example: From 50 to 75 = ((75-50)/50) × 100 = 50% increase

Finding What Percentage X is of Y

To find what percentage X is of Y:

Percentage = (X / Y) × 100

Example: 30 is what % of 120? (30/120) × 100 = 25%

Tips & Notes

  • To convert a percentage to a decimal, divide by 100 (e.g., 25% = 0.25).
  • To convert a decimal to a percentage, multiply by 100 (e.g., 0.75 = 75%).
  • When calculating percentage change, use the absolute value of the original number to handle negative values correctly.
  • A percentage increase followed by the same percentage decrease does not return to the original value.
  • Percentages can exceed 100% when the compared value is greater than the base value.
  • Negative percentages indicate a decrease or reduction from the original value.
